Frequently Asked Questions

Where are my API keys stored?

All API keys are stored locally on your device using iOS Keychain / Android Keystore. No data is sent to any server.

How do I delete my data?

Simply delete the app from your device. All data is removed since nothing is stored externally.

Does the app collect analytics?

No. API Matrix has no analytics, crash reporting, or telemetry of any kind.

How do I back up my API keys?

Go to Settings → Data Management → Export Backup. The backup JSON is copied to your clipboard.
