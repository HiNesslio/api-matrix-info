Privacy Policy

Last updated: June 13, 2026

Data Storage

API Matrix stores all user data exclusively on your device using the iOS Keychain and Android Keystore. No data is transmitted to external servers, cloud services, or third parties.

Information We Collect

We do not collect any personal information. The app does not have analytics, crash reporting, or any form of telemetry. Your API keys, usage data, and preferences remain solely on your device.

Advertising

API Matrix displays banner ads via Google AdMob using non-personalized ads only. No advertising ID (IDFA) is used, and no user profiling or behavioral targeting is performed. AdMob may serve contextual ads based on general location (country level) and app content.

Third-Party Services

The app communicates directly with third-party AI provider APIs only when you explicitly perform a ping test or fetch usage data. These requests are made using your own API keys and are not intercepted or logged by us.

Data Deletion

Since all data is stored locally, you can delete all data by deleting the app from your device. No data exists on any external server that would require a separate deletion request.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page.

Contact

If you have questions about this privacy policy, please contact us.
